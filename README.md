# 🎸 Band Club Database Project

저는 교내 밴드 동아리 회장으로 활동하며, 부원 관리와 공연 관리를 효율적으로 하기 위해 **밴드 동아리 데이터베이스**를 구축했습니다.  
동아리는 기수제로 운영되며, 각 기수마다 임원진과 세션장이 있으며, 부원들은 팀을 구성하여 공연에 참여합니다.  

---

## 📚 데이터베이스 설계

### 주요 테이블

| 테이블 | 설명 |
|--------|------|
| 부원 | 학번, 이름, 학과, 기수, 세션 정보 |
| 임원진 | 임원 정보 (학번, 기수, 직급) |
| 세션장 | 세션장 정보 (학번, 기수, 세션) |
| 팀 | 팀 번호, 곡 이름, 가수 |
| 공연 | 공연 번호, 행사명, 날짜, 지출 |
| 지원서 | 부원별 지원 정보 |
| 조언 | 부원 간 조언 관계 |
| 팀_부원 | 팀별 부원 구성 |
| 공연_팀 | 공연별 팀 참여 |


## 🗂 프로젝트 파일 구성

| 파일 | 설명 |
|------|------|
| DDL.sql | 테이블 및 스키마 생성 |
| insert.sql | 샘플 데이터 삽입 |
| 뷰.sql | 뷰(View) 생성 |
| 질의문.sql | 주요 SELECT 쿼리 |
| 트리거.sql | 트리거, 프로시저, 함수 |

> 각 파일에는 SQL 코드와 주석을 포함하여 상세한 내용을 확인할 수 있습니다.

---

## 💡 활용
- 동아리 부원 관리, 공연 일정 및 팀 구성 관리  
- 데이터 기반 통계 및 분석 가능
- 쿼리 예시를 통해 부원 활동 및 공연 참여를 쉽게 추적
